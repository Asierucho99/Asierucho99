## 📊 Proyectos de Data Science

### 📈 Análisis de Siniestralidad Laboral
Análisis de datos públicos de la Comunidad Valenciana (2019–2023) para detectar patrones de riesgo mediante **EDA, análisis estadístico y visualización de datos**.

### ⚽ Predictor de Resultados Deportivos
Sistema de **Machine Learning** capaz de predecir resultados de fútbol analizando variables históricas y métricas de rendimiento de los equipos.

### 🖼 Salvem les Fotos
Aplicación web construida con **Python y Streamlit** para gestionar procesos de restauración de imágenes dañadas utilizando herramientas de **IA aplicada a imagen**.
